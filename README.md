# Namma Santhe Ledger

## Overview
Namma Santhe Ledger is an Android application developed to help small vendors, local shopkeepers, and businesses manage customer transactions digitally. The application replaces traditional handwritten ledger books with a secure and easy-to-use mobile platform.

The app allows users to maintain customer records, track credit and debit transactions, monitor balances, and manage daily financial activities efficiently.

---

# Problem Statement
Small businesses often maintain financial records manually, which can lead to:
- Calculation errors
- Data loss
- Difficulty tracking payments
- Poor organization of customer records

Namma Santhe Ledger provides a digital solution for maintaining transaction records in a structured and secure way.
